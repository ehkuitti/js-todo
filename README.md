# JS Todo app 

JS todo app for storing tasks. Created using HTML, CSS, and JavaScript.

<img width="506" height="534" alt="kuva" src="https://github.com/user-attachments/assets/825f52c3-44e5-4548-a7a1-4ca19a8721d6" />

## Implemented features: 

- Add elements to list using enter
- Save list items to Localstorage
- Clear input field after an item waas added successfully to the list
- Check whether the input field is empty and show an alert on it

## Missing features (coming soon): 

- Remove items from the list
